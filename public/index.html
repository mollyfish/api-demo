An API demo

Posting Items

  • To post new lecture data, enter the following in the command line:

    superagent localhost:8080/api/lectures post '{"title":"lecture title", "subject":"lecture subject", "attendedBy":["student ID", "student ID" ... ]}'

  • To post new student data, enter the following in the command line:

    superagent localhost:8080/api/students post '{"name":"student name", "attended":["lecture ID", "lecture ID" ... ]}'

Getting Lecture Data

    • To get all lecture data, enter the following in the command line:

      superagent localhost:8080/api/lectures get

      Or enter this URL into the browser:

      localhost:8080/api/lectures

    • To get the title and subject of a certain lecture, enter the following in the command line:

      superagent localhost:8080/api/lectures/:id/subject get

      Or enter this URL into the browser:

      localhost:8080/api/lectures/:id/subject

    • To get the attendance of a certain lecture, enter the following in the command line:

      superagent localhost:8080/api/lectures/:id/attendance get

      Or enter this URL into the browser:

      localhost:8080/api/lectures/:id/attendance

    • To get student data, enter the following in the command line:

      superagent localhost:8080/api/students get

    Getting Student Data

    • To get all student data, enter the following in the command line:

      superagent localhost:8080/api/students get

      Or enter this URL into the browser:

      localhost:8080/api/students

    • To get the name of a certain student, enter the following in the command line:

      superagent localhost:8080/api/students/:id/name get

      Or enter this URL into the browser:

      localhost:8080/api/students/:id/name

    • To get the attendance of a certain student, enter the following in the command line:

      superagent localhost:8080/api/students/:id/attendance get

      Or enter this URL into the browser:

      localhost:8080/api/students/:id/attendance

    Editing Items

    • To edit lecture data, enter the following in the command line:

      superagent localhost:8080/api/lectures/lectureID put '{"data":"data"}'

    • To edit student data, enter the following in the command line:

      superagent localhost:8080/api/students/studentID put '{"data":"data"}'

    Deleting Items

    • To delete lecture data, enter the following in the command line:

      superagent localhost:8080/api/lectures/lectureID delete

    • To delete student data, enter the following in the command line:

      superagent localhost:8080/api/students/studentID delete
