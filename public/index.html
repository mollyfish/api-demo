An API demo

Posting Items

  • To post new lecture data, enter the following in the command line:

    superagent localhost:8080/api/lectures post '{"title":"lecture title", "subject":"lecture subject", "attendance":"number"}'

  • To post new student data, enter the following in the command line:

    superagent localhost:8080/api/students post '{"name":"student name", "attendance":"number"}'

Getting Items

  • To get lecture data, enter the following in the command line:

    superagent localhost:8080/api/lectures get

  • To get student data, enter the following in the command line:

    superagent localhost:8080/api/students get

Editing Items

  • To edit lecture data, enter the following in the command line:

    superagent localhost:8080/api/lectures/lectureID put '{"data":"data"}'

  • To edit student data, enter the following in the command line:

    superagent localhost:8080/api/students/studentID put '{"data":"data"}'

Deleting Items

  • To delete lecture data, enter the following in the command line:

    superagent localhost:8080/api/lectures/lectureID delete

  • To delete student data, enter the following in the command line:

    superagent localhost:8080/api/students/studentID delete
